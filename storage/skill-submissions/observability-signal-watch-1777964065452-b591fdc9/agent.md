# Observability Signal Watch

Observability Signal Watch helps teams evaluate whether their production telemetry is strong enough to detect and diagnose incidents quickly.

## What it checks

- Duplicate or noisy alerts
- Missing telemetry on critical flows
- Weak signal quality in dashboards
- Gaps between incidents and monitor coverage

## Supported stack

- Prometheus
- Grafana
- OpenTelemetry

## Usage

Run this skill when onboarding a service to production readiness reviews or after an incident where detection was delayed.

## Maintainers

- Site Reliability Engineering
- Monitoring Platform

## Expected output

- alert quality notes
- missing telemetry findings
- dashboard coverage recommendations
- incident detection improvement suggestions
