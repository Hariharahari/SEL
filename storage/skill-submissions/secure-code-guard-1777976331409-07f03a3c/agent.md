# Release Readiness Radar

Release Readiness Radar helps teams catch release blockers before deployment.

## What it checks

- CI pipeline status
- Required approvals
- Environment configuration
- Changelog completeness
- Rollback readiness

## Supported stack

- GitHub Actions
- Node.js
- YAML deployment workflows

## Usage

Run this skill before promoting a release candidate to staging or production.

## Maintainers

- Platform Reliability
- Release Operations

## Notes

Attach a short demo video and any supporting release checklist files when submitting this skill for review.
